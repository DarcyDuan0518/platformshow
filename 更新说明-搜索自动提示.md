## 📝 使用说明

### 🔍 搜索自动提示

#### 基础使用
1. **打开搜索侧边栏**：点击右上角 🔍 按钮
2. **输入关键词**：在搜索框输入姓氏或关键词
   - 输入 "钱" → 看到 钱学森
   - 输入 "航天" → 看到所有航天相关人物
   - 输入 "工匠" → 看到所有工匠
3. **选择建议**：
   - 方式1：点击建议项
   - 方式2：继续输入完整名字
4. **执行搜索**：
   - 点击建议 → 自动搜索
   - 点击 🔍 按钮
   - 按 `Enter` 键

#### 高级技巧
- **搜索姓氏**：输入 "钱" 可以找到所有姓钱的人
- **搜索领域**：输入 "医学" 找医学相关人物
- **搜索关键词**：输入 "诺贝尔" 找获奖人物
- **搜索标签**：输入 "两弹一星" 找相关人物

### 🎯 智能筛选

#### 即时筛选
选择任何筛选条件，结果**立即更新**：

1. **类别筛选**
   - 点击 [科学家] → 立即显示所有科学家
   - 点击 [工匠] → 立即显示所有工匠
   - 点击 [全部] → 显示所有人物

2. **领域筛选**
   - 选择 "航天工程" → 立即显示
   - 选择 "医学" → 立即显示

3. **年代筛选**
   - 选择 "当代" → 立即显示

#### 组合筛选
多个条件可以叠加使用：

**示例1：找当代的科学家**
```
1. 点击 [科学家]
2. 选择年代 "当代"
✓ 立即显示当代科学家
```

**示例2：搜索航天工匠**
```
1. 输入 "航天"
2. 点击 [工匠]
3. 点击 🔍 搜索
✓ 显示航天领域的工匠
```

### 🔄 清空筛选

点击 **"✕ 清空筛选"** 按钮：
- 清空搜索框
- 重置所有筛选条件
- 显示所有人物

## ❓ 常见问题

**Q: 输入后没有出现建议？**
A: 可能没有匹配的人物，试试输入姓氏或领域关键词。

**Q: 建议列表显示不全？**
A: 最多显示5条建议，输入更具体的关键词可以精确匹配。

**Q: 选择筛选条件后没反应？**
A: 现在是自动生效的，页面应该已经更新了，检查是否有匹配结果。

**Q: 如何同时搜索和筛选？**
A: 先输入搜索词，再选择筛选条件，所有条件会同时生效。

**Q: 建议列表怎么关闭？**
A: 点击页面其他地方、按Esc键、或者开始输入新内容。
