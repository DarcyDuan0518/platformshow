# 人物图片添加指南

## 📸 图片要求

### 基本要求
- **格式**: JPG 或 PNG
- **尺寸**: 建议宽度 600-1000px
- **质量**: 高清（至少 720p）
- **内容**: 半身照或正式照片
- **背景**: 干净整洁，最好是纯色或虚化背景

### 推荐规格
- **分辨率**: 800x1000px（竖版）
- **大小**: 200KB - 1MB
- **比例**: 4:5 或 3:4（竖版照片）
- **人物位置**: 人物在画面中上部，预留下方空间

## 🎨 图片处理建议

### 推荐图片编辑工具

#### Windows
- **Paint（画图）**: 系统自带，可以裁剪和调整大小
- **美图秀秀**: 免费，界面简单

#### MacBook
- **预览（Preview）**: 系统自带，可以裁剪
- **iPhoto / Photos**: 系统自带
- **Photoshop**: 专业工具

### 简单处理步骤

#### 1. 裁剪图片（保留人物半身）
```
- 打开图片
- 选择裁剪工具
- 框选人物半身部分（头部到胸部）
- 确保人物在画面中上部
- 保存
```

#### 2. 调整尺寸
```
推荐尺寸：800x1000px
最小尺寸：600x750px
最大尺寸：1200x1500px
```

#### 3. 优化质量
```
- 调整亮度：让人脸清晰可见
- 提高对比度：增强视觉效果
- 不要过度美化，保持真实
```

## 🔧 常见问题

### Q1: 图片不显示怎么办？

**检查步骤：**
1. 确认文件名是否正确（区分大小写）
2. 确认文件在 `images/` 文件夹内
3. 确认文件扩展名是 `.jpg` 或 `.png`
4. 尝试强制刷新页面 (Ctrl+F5)

**示例：**
```
❌ 错误: images/钱学森.JPG (扩展名大写)
✓ 正确: images/钱学森.jpg

❌ 错误: images/钱学森.jpeg
✓ 正确: images/钱学森.jpg

❌ 错误: 钱学森.jpg (不在 images 文件夹)
✓ 正确: images/钱学森.jpg
```

### Q2: 图片显示模糊？

**解决方案：**
- 下载更高分辨率的图片
- 确保图片宽度至少 600px
- 选择"大尺寸"图片

### Q3: 图片太大影响加载速度？

**优化方法：**
1. 使用在线压缩工具
   - TinyPNG: https://tinypng.com
   - Squoosh: https://squoosh.app
2. 压缩后大小控制在 200KB - 500KB

### Q4: 找不到合适的图片？

**替代方案：**
- 搜索"人物姓名 + 百度百科"
- 搜索"人物姓名 + 新华社"
- 搜索"人物姓名 + 官方照片"

### Q5: 图片位置不对（人物头部被遮挡）？

**解决方案：**
- 裁剪图片，让人物靠近上方
- 或者在CSS中调整 `background-position: center 20%;`
  可以改为 `center 10%;` 或 `center 30%;`
