## 🔧 视频转换工具

如果视频不是MP4格式，可以使用：

### 在线工具
- CloudConvert: https://cloudconvert.com/
- Online-Convert: https://www.online-convert.com/

### 桌面软件
- **HandBrake** (免费): https://handbrake.fr/
- **Format Factory** (免费): http://www.pcfreetime.com/
- **FFmpeg** (命令行): https://ffmpeg.org/

### FFmpeg 快速转换命令
```bash
# 转换为MP4并压缩
ffmpeg -i 输入视频.avi -c:v libx264 -crf 23 -c:a aac -b:a 128k 输出视频.mp4

# 调整为720p
ffmpeg -i 输入.mp4 -vf scale=-1:720 -c:v libx264 -crf 23 输出.mp4
```

## ⚠️ 注意事项

1. **文件名必须完全匹配**
   - 区分大小写
   - 包含正确的扩展名 (.mp4)
   - 不要有多余的空格

2. **视频编码兼容性**
   - 推荐 H.264 编码（最广泛支持）
   - 避免使用专有编码格式

3. **文件大小控制**
   - 单个视频文件不要超过500MB
   - 建议控制在50-200MB之间

4. **版权合规**
   - 仅用于教育、展览等非商业用途
   - 建议使用官方发布或授权的视频

## 🐛 故障排除

### 视频无法播放？
1. 检查文件名是否完全正确
2. 检查视频格式（必须是MP4）
3. 打开浏览器控制台查看错误信息（F12）
4. 尝试用其他播放器播放该视频文件，确认文件本身没有损坏

### 视频加载很慢？
1. 检查视频文件大小，建议压缩
2. 降低视频分辨率到720p
3. 使用更高的压缩比（CRF值调高）

### 手机上无法播放？
1. 确认使用H.264编码
2. 避免使用过高的分辨率
3. 测试不同的移动浏览器
