## 📸 图片要求

### 基本要求
- **格式**: JPG 或 PNG
- **尺寸**: 600-1000px 宽度
- **比例**: 竖版（3:4 或 4:5）
- **内容**: 半身照或正式照片
- **质量**: 高清（720p以上）

### 推荐规格
```
分辨率: 800x1000px
大小: 200KB - 1MB
人物位置: 画面中上部
背景: 干净整洁
```

### 文件命名规则
```
格式: 人物姓名.jpg
示例: 钱学森.jpg
```

**注意**：
- 使用中文命名
- 扩展名小写
- 与人物姓名完全一致

## 🔍 如何获取图片

### 快速步骤

#### 1. 打开搜索引擎
```
推荐: 百度图片、必应图片、百度百科
```

#### 2. 搜索人物照片
```
搜索词格式: 人物姓名 + 照片 + 高清
示例: 钱学森 照片 高清
```

#### 3. 选择合适图片
```
- 点击"大尺寸"筛选
- 选择半身正式照片
- 避免全身照（人物太小）
```

#### 4. 保存图片
```
- 右键 → "图片另存为"
- 保存到 images/ 文件夹
- 命名为 "人物姓名.jpg"
```

## 💡 使用技巧

### 图片来源优先级
1. **百度百科 / 维基百科** - 质量最高
2. **新华社 / 人民日报** - 官方照片
3. **百度图片大尺寸** - 筛选高清图
4. **必应图片** - 国际来源

### 图片筛选技巧
- 使用"大尺寸"筛选器
- 选择官方、正式场合照片
- 避免过度美化的图片
- 优先选择半身照

4. 添加渐变遮罩确保文字可读

## ❓ 常见问题

### Q: 图片不显示？
**检查**：
- 文件名是否正确（钱学森.jpg）
- 文件是否在 images/ 文件夹
- 扩展名是否小写 (.jpg)
- 尝试强制刷新 (Ctrl+F5)

### Q: 图片模糊？
**解决**：
- 下载更高分辨率的图片
- 确保宽度至少 600px
- 选择"大尺寸"图片

### Q: 图片太大？
**优化**：
- 使用在线压缩工具 (TinyPNG)
- 控制大小在 200KB - 500KB

### Q: 人物头部被遮挡？
**调整**：
- 裁剪图片，让人物靠上
- 或修改CSS的 `background-position`
