## 添加/编辑人物教程

### 📍 文件位置
编辑文件：`科学家工匠展示平台/data.js`

---

### ➕ 添加新人物

在 `data.js` 文件中，找到最后一个人物数据（邓稼先），在他的 `}` 后面添加逗号和新人物：

```javascript
    {
        id: 8,  // 邓稼先的数据
        ...
    },  // ← 确保这里有逗号！
    {
        id: 9,  // ← 新人物从这里开始
        name: "新人物姓名",
        category: "scientist",  // scientist(科学家) 或 craftsman(工匠)
        field: "所属领域",
        era: "当代",  // 近代/现代/当代
        birth: "1950",
        death: "2020",  // 如果健在，可以省略这行
        image: "",
        shortDesc: "一句话简介，显示在卡片上，50-80字为宜。",
        fullDesc: "详细介绍，200-500字，介绍生平、主要贡献等。",

        // 时间线（3-7个关键节点）
        timeline: [
            {
                year: "1950",
                title: "出生",
                desc: "简单描述"
            },
            {
                year: "1975",
                title: "大学毕业",
                desc: "毕业于某某大学"
            },
            {
                year: "2000",
                title: "重大成就",
                desc: "取得什么成就"
            }
        ],

        // 主要成就（3-5项）
        achievements: [
            {
                title: "成就标题",
                desc: "详细描述这项成就的内容和意义"
            },
            {
                title: "第二个成就",
                desc: "描述..."
            }
        ],

        // 精神品质（4项，固定4项效果最好）
        spirit: [
            {
                icon: "💡",  // 选择合适的emoji图标
                name: "品质名称",
                desc: "简短描述这个品质"
            },
            {
                icon: "🔬",
                name: "第二个品质",
                desc: "描述..."
            },
            {
                icon: "🏆",
                name: "第三个品质",
                desc: "描述..."
            },
            {
                icon: "❤️",
                name: "第四个品质",
                desc: "描述..."
            }
        ],

        // 标签（3-5个）
        tags: ["标签1", "标签2", "标签3", "标签4"]
    }
];  // ← 注意：最后一个人物后面不要逗号
```

### ✏️ 修改现有人物

直接在 `data.js` 中找到对应人物的数据，修改即可：

```javascript
{
    id: 1,
    name: "钱学森",  // ← 直接修改这里的内容
    category: "scientist",
    shortDesc: "修改简介...",  // ← 修改简介
    // 其他字段同理
}
```

---

### ❌ 删除人物

找到要删除的人物数据，删除整个 `{ }` 块：

```javascript
// 删除前：
{
    id: 1,
    name: "钱学森",
    ...
},  // ← 删除从这里开始
{
    id: 2,
    name: "袁隆平",  // ← 到这里之前
    ...
}

// 删除后：
{
    id: 2,
    name: "袁隆平",
    ...
}
```

**注意**：删除后要调整ID，保证连续（可选，不连续也能工作）

### 📋 字段说明

| 字段 | 必填 | 说明 | 示例 |
|------|------|------|------|
| `id` | ✅ | 唯一标识符，数字 | 1, 2, 3... |
| `name` | ✅ | 姓名 | "钱学森" |
| `category` | ✅ | 类别 | "scientist" 或 "craftsman" |

| `death` | ❌ | 去世年份 | "2009"（健在可省略） |
| `image` | ❌ | 图片URL | 暂时留空"" |
| `shortDesc` | ✅ | 卡片简介 | 50-80字 |
| `fullDesc` | ✅ | 详细介绍 | 200-500字 |
| `timeline` | ✅ | 时间线数组 | 3-7个节点 |
| `achievements` | ✅ | 成就数组 | 3-5项 |
| `spirit` | ✅ | 精神品质数组 | 固定4项 |
| `tags` | ✅ | 标签数组 | 3-5个 |

### ⚠️ 常见错误

#### 1. 缺少逗号
```javascript
// ❌ 错误：两个对象之间缺少逗号
{
    id: 1,
    name: "钱学森"
}
{
    id: 2,
    name: "袁隆平"
}

// ✅ 正确
{
    id: 1,
    name: "钱学森"
},  // ← 这里要有逗号
{
    id: 2,
    name: "袁隆平"
}
```

#### 2. 多余的逗号
```javascript
// ❌ 错误：最后一个元素后面有逗号
{
    id: 8,
    name: "邓稼先"
},  // ← 最后一个不要逗号
]
```

#### 3. 引号错误
```javascript
// ❌ 错误：使用中文引号
name: "钱学森"

// ✅ 正确：使用英文引号
name: "钱学森"
```

#### 4. 字段拼写错误
```javascript
// ❌ 错误
categary: "scientist"  // 拼写错误

// ✅ 正确
category: "scientist"
```

### 💡 实用技巧

#### 1. 复制模板
直接复制一个现有人物的完整数据，然后修改内容

#### 2. 保持格式统一
- 时间线：3-7个节点最佳
- 成就：3-5项最佳
- 精神品质：固定4项
- 标签：3-5个最佳

#### 3. 文字长度建议
- **shortDesc**：50-80字，显示在卡片上
- **fullDesc**：200-500字，显示在详情页
- **时间线desc**：10-30字
- **成就desc**：30-60字
- **精神desc**：10-20字

#### 4. 选择合适的emoji
- 每个领域选择代表性的emoji
- 精神品质用表达情感的emoji
- 避免使用过于复杂的emoji

### 🆘 遇到问题？

#### 页面空白或显示错误
1. 按 `F12` 打开开发者工具
2. 点击 "Console" (控制台)
3. 查看红色错误信息
4. 常见错误：
   - `Unexpected identifier`: 引号或逗号错误
   - `Unexpected token`: 语法错误
   - `peopleData is not defined`: 文件加载失败

#### 数据不显示
1. 检查 `category` 字段是否正确（只能是 "scientist" 或 "craftsman"）
2. 检查所有必填字段是否都填写了
3. 检查引号是否都是英文引号

#### 格式混乱
1. 检查 `timeline`、`achievements`、`spirit` 数组格式
2. 确保每个对象都用 `{}` 包裹
3. 确保数组元素之间有逗号
