# 科学家与行业匠人精神展示平台

一个专为大学生打造的视觉展示平台，用于展现科学家和行业匠人的感人事迹和精神品质。

## ✨ 平台特色

### 🎯 设计理念
- **展示为主** - 大屏横幅、精美卡片、视觉冲击力强
- **分区展示** - 科学家和工匠分区域展示，层次清晰
- **特色突出** - 首页特色人物大卡片展示
- **搜索隐藏** - 搜索筛选功能放在侧边栏，不干扰浏览

### 💫 视觉效果
- 渐变色背景和卡片设计
- 流畅的动画和过渡效果
- 响应式布局，适配各种设备
- 滚动加载动画

### 📱 功能特性
- ⭐ 首页特色人物展示
- 🔬 科学家专区展示
- 🔧 行业匠人专区展示
- 💫 精神传承专题展示
- 🔍 侧边栏搜索筛选（按需使用）
- 📖 详细人物事迹页面
- ⏰ 可视化时间线
- 📊 动态统计数字

## 🚀 使用方法

### 方法一：直接打开（最简单）
1. 进入文件夹
2. 双击 `index.html`
3. 浏览器自动打开

### 方法二：使用本地服务器（推荐）

#### Windows系统
双击 `start-server.bat` 文件，浏览器会自动打开

#### Mac/Linux系统
```bash
cd 科学家工匠展示平台
python3 -m http.server 8000
# 然后访问 http://localhost:8000
```

## 💡 如何添加新人物

编辑 `data.js` 文件，在 `peopleData` 数组末尾添加：

```javascript
{
    id: 9,                      // 唯一ID（递增）
    name: "姓名",
    category: "scientist",      // scientist 或 craftsman
    field: "领域",
    era: "当代",                // 近代/现代/当代
    birth: "1950",
    death: "2020",              // 如健在可不填
    image: "",
    shortDesc: "一句话简介（显示在卡片上）",
    fullDesc: "详细介绍（显示在详情页）...",
    timeline: [                  // 人生历程时间线
        {
            year: "1950",
            title: "事件标题",
            desc: "事件描述"
        }
    ],
    achievements: [              // 主要成就
        {
            title: "成就标题",
            desc: "成就详细描述"
        }
    ],
    spirit: [                    // 精神品质（4项）
        {
            icon: "💡",         // emoji图标
            name: "品质名称",
            desc: "品质描述"
        }
    ],
    tags: ["标签1", "标签2", "标签3"]
}
```

## 🔧 自定义修改

### 修改颜色主题
编辑 `styles.css` 顶部的 CSS 变量：
```css
:root {
    --primary: #667eea;        /* 主色 */
    --secondary: #764ba2;      /* 辅色 */
    --accent: #f093fb;         /* 强调色 */
}
```

### 修改特色人物
编辑 `script.js` 的 `renderFeaturedPeople()` 函数：
```javascript
const featured = peopleData.slice(0, 3); // 修改数字选择不同人物
```

### 添加图片背景
为人物卡片添加图片，在 `data.js` 中填写 `image` 字段，然后修改CSS。
