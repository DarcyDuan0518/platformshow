## 🎬 视频要求

### 文件格式
- **推荐格式**：MP4 (H.264编码)
- **备选格式**：WebM, OGG
- **不推荐**：AVI, WMV（部分浏览器不支持）

### 文件大小
- **建议大小**：50MB - 200MB 之间
- **时长建议**：3-10分钟
- 太大的视频文件会影响加载速度

### 视频质量
- **分辨率**：1280x720 (720p) 或 1920x1080 (1080p)
- **码率**：建议 2-5 Mbps
- **音频**：AAC编码，128-256 kbps

## 🔧 视频格式转换

如果您的视频不是MP4格式，可以使用以下工具转换：

### 在线工具
- [CloudConvert](https://cloudconvert.com/video-converter) - 在线转换，无需安装
- [Online-Convert](https://www.online-convert.com/) - 支持批量转换

### 桌面软件
- **HandBrake** (免费，跨平台) - https://handbrake.fr/
- **Format Factory** (免费，Windows) - http://www.pcfreetime.com/
- **FFmpeg** (命令行工具) - https://ffmpeg.org/

### FFmpeg 转换命令示例
```bash
# 转换为MP4并压缩
ffmpeg -i 输入视频.avi -c:v libx264 -crf 23 -c:a aac -b:a 128k 输出视频.mp4

# 调整分辨率为720p
ffmpeg -i 输入视频.mp4 -vf scale=-1:720 -c:v libx264 -crf 23 -c:a copy 输出视频.mp4
```

## 🔄 如果暂时没有视频

如果某个人物暂时没有视频文件：

1. **方式一**：在 data.js 中将该人物的 `video` 字段设为空字符串
   ```javascript
   video: "",  // 暂无视频
   ```

2. **方式二**：使用占位视频
   - 可以准备一个通用的"视频即将上线"占位视频

3. **方式三**：添加外部链接
   - 在详情页添加"观看在线视频"按钮，链接到B站等平台

## ⚠️ 常见问题

### Q: 视频无法播放？
A: 检查：
- 文件名是否完全正确（区分大小写）
- 文件格式是否为MP4
- 视频编码是否为H.264
- 浏览器控制台是否有错误信息

### Q: 视频加载很慢？
A: 解决方法：
- 压缩视频文件大小
- 降低视频分辨率（720p足够）
- 使用更高效的编码参数

### Q: 手机上无法播放？
A: 可能原因：
- 移动浏览器对某些编码格式支持不好
- 建议使用标准的H.264 + AAC编码
