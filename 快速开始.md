# 快速开始指南

## 🚀 3步开始使用

### 第1步：打开文件夹
找到 `科学家工匠展示平台` 文件夹

### 第2步：启动平台
**最简单的方法：**
- 双击 `index.html` 文件即可在浏览器中打开

**推荐方法（更好的体验）：**
- 双击 `start-server.bat` 文件
- 浏览器会自动打开平台

### 第3步：开始浏览
- 向下滚动查看所有人物
- 点击人物卡片查看详细信息
- 点击右上角 🔍 按钮进行搜索和筛选

## 📖 页面导航

1. **首页横幅** - 展示平台主题和统计信息
2. **特色人物** - 3位重点人物大卡片展示
3. **科学家专区** - 浏览所有科学家事迹
4. **大国工匠专区** - 浏览所有工匠事迹
5. **精神传承** - 了解四大核心精神品质

## 🔍 搜索功能

- 点击右上角的 🔍 圆形按钮
- 侧边栏滑出，可以：
  - 输入关键词搜索
  - 按类别筛选（全部/科学家/工匠）
  - 按领域筛选
  - 按年代筛选
  - 点击"清空筛选"重置

## 💡 查看详情

点击任意人物卡片，弹出详情页面，包含：
- 📖 完整人物简介
- ⏰ 人生历程时间线
- 🏆 主要成就列表
- ✨ 精神品质展示
- 🏷️ 相关标签

按 ESC 键或点击背景关闭详情页

## ⚙️ 系统要求

- Windows 7 及以上
- 现代浏览器（Chrome、Edge、Firefox、Safari）
- 如使用启动脚本需要Python环境（大多数电脑已安装）

## ❓ 常见问题

**Q: 双击 start-server.bat 没反应？**
A: 可能没有安装Python，直接双击 index.html 即可

**Q: 页面显示不正常？**
A: 请使用现代浏览器，建议使用Chrome或Edge

**Q: 如何添加新人物？**
A: 编辑 data.js 文件，参考现有格式添加

**Q: 能不能修改颜色？**
A: 可以！编辑 styles.css 文件顶部的颜色变量

## 📱 手机访问

使用启动脚本后，手机连接同一WiFi：
1. 查看电脑IP地址（cmd输入 ipconfig）
2. 手机浏览器访问 http://你的IP:8000
